변수의 명명법

  • 변수의 이름은 영문자(대소문자)와 숫자, '_','$'만 사용
  • 변수의 시작은 숫자를 제외한 나머지로 시작
  • 변수의 이름에 2 단어 이상이 합쳐질 경우 Camel Expression 적용
  • 변수명은 다른 사람이 보더라도 변수의 의미를 이해할 수 있는 이름을 붙임
  • 예약어(var, let, null, undefined, number, for, if ...)는 제외

변수의 할당(선언+초기화)

  • num = 10 // 암묵적 전역공간에 변수가 생겨서 진행시 충돌 가능성(실무 비추)
  • var num = 10 // 의도하여 선언하는 방식
  • let num = 10
  • const num = 10
    • 제어자(Modifier) 재선언 재할당 Scope(범위)
    var ok ok 함수 scope
    let no ok 블록 scope
    const no no 블록 scope
