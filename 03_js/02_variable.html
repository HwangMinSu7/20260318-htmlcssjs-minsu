변수의 타입

  1. 원시타입(primitive Type)
    2. 값 자체가 메모리에 직접 저장, 7가지
    • number : 숫자(정수, 실수)
    • bigint : 숫자인데 매우 큰 수
    • string : 문자열
    • boolean : true, false
    • undefined : 값이 초기화 되지 않을 떄 값을 출력
  2. null : 값이 없음을 의도적으로 명시할 때 사용 / 타입은 object
  3. symbol : 절대 중복되지 않는 고유한 식별자를 만들때 사용(ES6)
  4. 참조타입(Referenced Type)
    5. 주소값, 주소에 가면 집합군에 접근
    • object
    • Array
    • function
    • DOM, BOM
    • 내장 객체 (date, math, ...)
    • 변수의 형변환
      • var, let, 제어자 없이 선언해도 타입의 변환이 자유롭다.
